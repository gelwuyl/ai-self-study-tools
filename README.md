# 🧠 AI Self-Study Tools for Data Science Learners

> A beginner-friendly guide to using AI tools to help you survive — and thrive — in a technical data science and AI course, even without a tech background.

---

## Who Is This For?

Adult learners entering a data science or AI course for the first time. No coding experience required. This guide helps you use AI tools to **read research papers, understand complex concepts, and ask better questions**.

---

## 📋 Lesson Plan

**Duration:** 1 – 1.5 hours  
**Format:** Instructor-led with hands-on practice  
**Tools needed:** Google account, access to a browser

| Time | Module | What You'll Do |
|------|--------|----------------|
| 0:00 – 0:05 | Welcome | Overview and goals |
| 0:05 – 0:40 | [Module 1: NotebookLM](lessons/01-notebooklm.md) | Upload a research paper, explore AI-powered reading tools |
| 0:40 – 1:15 | [Module 2: Prompting](lessons/02-prompting.md) | Learn 3 techniques for writing prompts that actually work |
| 1:15 – 1:20 | Wrap-up | Reflection and next steps |

## 🎯 Learning Outcomes

By the end of this session, you will be able to:

1. Upload a research paper into NotebookLM and use it to generate summaries, ask questions, and create audio overviews
2. Write a clear, structured prompt using the Gemini Prompt 101 framework
3. Use the "Who is the expert?" technique to get expert-level explanations from AI
4. Use a Meta Prompt to let AI help you write better prompts
